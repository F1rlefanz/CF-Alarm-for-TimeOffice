⚙️ Erweiterte Konfiguration

Detaillierte Anleitung zur optimalen Einrichtung von CF Alarm for Time Office

📅 Google Calendar Setup

🔑 OAuth 2.0 Konfiguration

  • Google Cloud Console: Projekt erstellen und Calendar API aktivieren
  • OAuth Client: Android-Anwendung registrieren
  • SHA-1 Fingerprint: Debug- und Release-Keys hinzufügen

🏂 TimeOffice Integration

  • Kalender freigeben: TimeOffice → Einstellungen → Google Kalender aktivieren
  • Synchronisation: Automatische Sync-Intervalle konfigurieren
  • Kategorien: Schichttypen (Früh/Spät/Nacht) definieren

💡 Philips Hue Konfiguration

🌉 Bridge Setup

  • Netzwerk: Bridge und Smartphone im gleichen WLAN
  • Bridge-Button: Physischen Button vor Koppelung drücken
  • Auto-Discovery: App findet Bridge automatisch

🌈 Licht-Szenarien

  • Morgen-Simulation: Graduelle Helligkeitssteigerung
  • Farbtemperatur: Warmes Licht am Morgen, kalt am Tag
  • Timing: 15 Minuten vor Alarmzeit starten

⚙️ Erweiterte Einstellungen

🔋 Power Management

  • Batterieoptimierung: CF Alarm von Optimierung ausschließen
  • Autostart: App-Start beim Systemboot aktivieren
  • Background: Hintergrund-Verarbeitung erlauben
💡 Pro-Tipp: Für optimale Performance alle Berechtigungen gewähren und Batterieoptimierung deaktivieren.
