🤝 Mitwirken

Werden Sie Teil der CF Alarm Community und helfen Sie bei der Entwicklung mit!

🎆 Wie Sie beitragen können

📝 Dokumentation

  • README verbessern: Rechtschreibung, Klarheit, Beispiele
  • Übersetzungen: Hilfe bei der Lokalisierung in andere Sprachen
  • Tutorials: Schritt-für-Schritt Anleitungen erstellen

🐛 Bug Reports

  • Issues erstellen: Detaillierte Fehlerbeschreibungen
  • Reproduzierbarkeit: Schritte zur Nachstellung bereitstellen
  • Logs: Relevante Log-Ausgaben beifügen

✨ Feature Requests

  • Use Cases: Erklären Sie, warum das Feature nützlich ist
  • Mockups: Visuelle Entwürfe sind willkommen
  • Community: Diskutieren Sie Ideen in den Discussions

💻 Code Contributions

  • Pull Requests: Folgen Sie unseren Coding Standards
  • Tests: Fügen Sie Unit Tests für neue Features hinzu
  • Code Review: Helfen Sie bei der Überprüfung von PRs

🛠️ Development Setup

# Repository klonen
git clone https://github.com/F1rlefanz/CF-Alarm-for-TimeOffice.git

# Branch erstellen
git checkout -b feature/mein-feature

# Änderungen committen
git commit -m "feat: neue Funktion hinzufügen"

# Pull Request erstellen
git push origin feature/mein-feature
