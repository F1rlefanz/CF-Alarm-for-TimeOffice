💻 Entwickler-Guide

Technische Dokumentation für Entwickler und erweiterte Benutzer

🚀 Erste Schritte

📋 Voraussetzungen

  • Android Studio: Arctic Fox oder neuer
  • Kotlin: 1.9.0+
  • Min SDK: API Level 26 (Android 8.0)
  • Target SDK: API Level 34 (Android 14)

🔧 Setup

git clone https://github.com/F1rlefanz/CF-Alarm-for-TimeOffice.git
cd CF-Alarm-for-TimeOffice
./gradlew build

🏗️ Architektur

📱 MVVM Pattern

  • Model: Datenmodelle und Repository Pattern
  • View: Activities/Fragments mit Data Binding
  • ViewModel: Business Logic und UI State Management

🗂️ Module Struktur

  • app: Haupt-Android-Modul
  • core: Gemeinsame Utilities und Extensions
  • data: Repository, Database, Network
  • domain: Use Cases und Business Logic
